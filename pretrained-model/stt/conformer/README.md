# Conformer

Conformer + RNNT.

## how-to

1. Run training script, 

SMALL size,

```bash
python3 base-conformer.py
```

BASE size,

```bash
python3 small-conformer.py
```

LARGE size,

```bash
python3 large-conformer.py
```
